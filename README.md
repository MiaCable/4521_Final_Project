# 4521_Final_Project: 
A Jupiter Notebook Feed Forward Neural Network Classification model written in Python, built to predict cricket match winners based on an augmented data set. 

## Key Features in augmented data set 
* Team 1 Win Rate
* Team 2 Win Rate
* Team 1 Head-to-Head Ratio
* Team 2 Head-to-Head Ratio
* Home Country
* Team 1 Captian Win Rate
* Team 2 Captian Win Rate
* Team 1 Captain Match Count
* Team 2 Captian Match Count 

## How to Run: 
Navigate to the 4521_Final_Project directory in your terminal. Then run the following, 
1. pip install jupyter
2. jupyter notebook Final_Project.ipynb
